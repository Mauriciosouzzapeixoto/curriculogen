# Professional CV Generator - Documentação Funcional

## 📋 Visão Geral

O **Professional CV Generator** é uma aplicação web completa para geração de currículos profissionais com foco em inclusão e acessibilidade para PcD (Pessoas com Deficiência). O sistema oferece temas personalizados por área profissional e garante a visibilidade e destaque da seção PcD em todos os contextos.

---

## 🛡️ Regras de Ouro do Sistema

### ♿ **Regra PcD #1: Visibilidade Obrigatória**
```
A seção PcD NUNCA pode ser ocultada, removida ou minimizada.
Deve manter destaque visual em TODOS os temas.
Dados PcD são preservados em TODAS as operações.
```

### 🔄 **Regra #2: Persistência de Dados**
```
Todos os dados são salvos automaticamente.
Tema selecionado é mantido ao recarregar.
Dados PcD têm prioridade máxima.
Nenhuma operação pode perder dados PcD.
```

### 🎨 **Regra #3: Adaptabilidade Temática**
```
Cada área profissional tem seu tema específico.
Cores se adaptam automaticamente.
Skills são contextualizadas por área.
Experiências são relevantes à área escolhida.
```

## 🚀 Como Usar o Sistema

### 1. **Setup Inicial**
1. Abra o sistema em um navegador moderno
2. O sistema carregará com tema padrão azul marinho
3. Dados salvos serão recuperados automaticamente

### 2. **Seleção de Área**
1. Escolha sua área profissional no seletor
2. Confirme a carga de sugestões contextuais
3. O tema visual se adaptará automaticamente

### 3. **Preenchimento de Dados**
1. Complete dados pessoais (sincronização automática)
2. Adicione experiências profissionais
3. Inclua habilidades técnicas relevantes

### 4. **Configuração PcD (Opcional mas Recomendado)**
1. Marque "Perfil Profissional PcD"
2. Preencha CID, necessidades e links
3. A seção terá destaque especial no currículo

### 5. **Geração do Currículo**
1. Revise o preview em tempo real
2. Use "Gerar PDF" para impressão
3. O formato A4 é otimizado para impressão
